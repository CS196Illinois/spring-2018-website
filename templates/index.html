GitHub

GitHub is a development platform inspired by the way you work. From open source to business, you can host and review code, manage projects, and build software alongside millions of other developers.

Piazza

An incredibly easy, completely free Q&A platform. Have a question? Post on Piazza! But for real though, try to search if someone else had the same question first before posting. Thanks!

Relate

We use RELATE to do the first few homeworks. Don't procrastinate on them! They can get a little tricky at times, so start early.

Stack Overflow

Stack Overflow is the largest, most trusted online community for developers to learn, share​ ​their programming ​knowledge, and build their careers. You could probably get your CS degree from this website.
